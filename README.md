# Web Crawler

This is a REST Interface for web crawler
# Setup pipenv
```bash
pipenv shell
pipenv install
````

## Run the Project
to run server

```python
export FLASK_APP=wsgi.py
export FLASK_RUN_PORT=8001 #only if you want to run it on different port
flask run
```
